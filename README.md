# GraphQL-Nodejs

Process to setup : 
- create a keys.js
- add your connection id just like the fomrat of file keys_sample.js
- RUN : npm i --save
- RUN : npm start

Initially it isnt using mondodb and is woorking on static data to go with the graphQL structure.
